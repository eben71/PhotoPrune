# Do Not Build

These items are forbidden or deferred unless an explicit future task changes scope and updates MVP criteria first.

- Do not build auto-delete.
- Do not build write-scope Google Photos actions.
- Do not build numeric similarity percentages unless a dedicated product-policy task explicitly resolves that decision and updates MVP criteria, trust docs, tests, and UI copy rules together.
- Do not build full-library scanning.
- Do not build account/settings pages beyond required MVP account details and settings.
- Do not build recovery/trash/recently-deleted flows.
- Do not add unsupported claims about local-only processing, privacy guarantees, storage reclaimed, or deletion safety.
- Do not expand architecture or persistence unless requested by a task.
- Do not add new dependencies unless task-justified.
